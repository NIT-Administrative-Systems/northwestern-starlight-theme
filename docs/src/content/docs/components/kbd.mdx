Render keyboard shortcuts as styled key caps. Parses `+`-separated combinations, maps key names to symbols (⌘, ⇧, ⌥), and swaps modifiers per platform.

## Import

```mdx
import { Kbd } from "@nu-appdev/northwestern-starlight-theme/components";
```

## Basic Usage

Pass key combinations as children, separated by `+`.

<Demo>
    Copy: <Kbd>Ctrl+C</Kbd> · Paste: <Kbd>Ctrl+V</Kbd> · Undo: <Kbd>Ctrl+Z</Kbd>
</Demo>

```mdx
<Kbd>Ctrl+C</Kbd>
<Kbd>Ctrl+V</Kbd>
<Kbd>Ctrl+Z</Kbd>
```

## Platform-Aware Modifiers

Use `Mod`, `Command`, `Ctrl`, `Alt`, or `Option` in the key string. Platform is detected automatically — Mac users see ⌘, Windows/Linux users see Ctrl.

<Demo>
    Open search: <Kbd>Mod+K</Kbd> · Command palette: <Kbd>Mod+Shift+P</Kbd> · Inspector: <Kbd>Alt+Shift+I</Kbd>
</Demo>

```mdx
<Kbd>Mod+K</Kbd>
<Kbd>Mod+Shift+P</Kbd>
<Kbd>Alt+Shift+I</Kbd>
```

## Key Symbol Mapping

Key names map to standard symbols. Unrecognized names render as-is (<Kbd>A</Kbd>, <Kbd>F12</Kbd>). Modifier symbols change per platform.

## With Description

The `tip` prop wraps the shortcut in a [`<Tooltip/>`](/components/tooltip), showing a description on hover.

<Demo>
    <Kbd tip="Open command palette">Mod+Shift+P</Kbd>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<Kbd tip="Save file">Mod+S</Kbd>
</Demo>

```mdx
<Kbd tip="Open command palette">Mod+Shift+P</Kbd>
<Kbd tip="Save file">Mod+S</Kbd>
```

## Explicit Keys Array

Use the `keys` array to specify each key explicitly.

<Demo>
    <Kbd keys={["Ctrl", "Shift", "I"]} />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<Kbd keys={["Command", "K"]} />
</Demo>

```mdx
<Kbd keys={["Ctrl", "Shift", "I"]} />
<Kbd keys={["Command", "K"]} />
```

## `<Kbd>` Props

**Implementation:** [`Kbd.astro`](https://github.com/NIT-Administrative-Systems/northwestern-starlight-theme/blob/main/packages/northwestern-starlight-theme/src/components/Kbd.astro)

<PropertyTable>
    <Property name="children" type="string">
        Key combination string. Parsed on `+` into individual keys. Key names are mapped to symbols.
    </Property>
    <Property name="keys" type="string[]">
        Explicit array of key names. Overrides children parsing.
    </Property>
    <Property name="tip" type="string">
        Description shown on hover via Tooltip.
    </Property>
</PropertyTable>
