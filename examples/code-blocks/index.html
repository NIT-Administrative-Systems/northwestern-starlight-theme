Code Blocks

Overview

Section titled “Overview”

The theme ships with Northwestern code block styling. This page covers the Code component patterns most useful when authoring docs.

Terminal Frames

Section titled “Terminal Frames”

Use the default terminal presentation for commands and shell output.

Code Frames

Section titled “Code Frames”

frame="code" renders the snippet as a file instead of a terminal command.

If your site includes an ec.config.mjs file using northwesternExpressiveCode(), line numbers are available as an opt-in feature.

Add showLineNumbers to a fenced code block when you want a numbered gutter:

In the <Code> component, use the showLineNumbers prop directly.

Frame Variants

Section titled “Frame Variants”

frame="none" removes the frame for quieter inline examples.

Markers

Section titled “Markers”

Markers highlight specific lines or text ranges.

Wrapping

Section titled “Wrapping”

Long snippets can wrap cleanly with wrap, preserveIndent, and hangingIndent.

Class Hook

Section titled “Class Hook”

The class prop applies a one-off style variation to a specific example.
